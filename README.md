# Customer Journey Analytics using Machine Learning

## Overview

This project analyzes large-scale customer journey and clickstream data from an e-commerce platform to understand the factors that drive customer purchases, account activation, and engagement behavior.

The objective was to identify high-impact customer interactions, model customer conversion pathways, and generate actionable business insights using machine learning, sequence analysis, and explainable AI techniques.

---

## Objectives

* Predict customer purchase behavior
* Predict account activation outcomes
* Identify influential customer journey events
* Discover optimal conversion pathways
* Generate actionable business recommendations

---

## Dataset

The dataset consists of customer clickstream and interaction records collected from an e-commerce platform, including:

* Website interactions
* Customer journey events
* Purchase activities
* Account activation events
* Marketing engagement behaviors

The raw event data was transformed into structured customer-level features for predictive modeling and behavioral analysis.

---

## Methodology

### Data Preparation

* Data cleaning
* Duplicate removal
* Event aggregation
* Feature engineering
* Customer journey transformation

### Machine Learning Models

Implemented and evaluated:

* XGBoost
* Random Forest
* Logistic Regression
* Recurrent Neural Networks (RNN)

### Behavioral Analytics

Implemented:

* Apriori Association Rule Mining
* Markov Chain Analysis

to discover common customer pathways and behavioral patterns.

### Explainable AI

Applied:

* SHAP (SHapley Additive exPlanations)

to identify the most influential events driving purchases and account activation.

---

## Key Results

* Developed high-performing predictive models for customer conversion and activation.
* Identified critical behavioral events associated with purchases and account activation.
* Discovered common customer journey patterns using sequence analysis techniques.
* Generated data-driven recommendations for marketing optimization and customer onboarding.
